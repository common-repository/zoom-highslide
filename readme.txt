== Description ==

This is a zoom and highslide plugin for wordpress, it works mainly as highslide .

[Demo](http://ziming.org/photo)

== Installation ==

1. Unzip the archive and put the 'zoom-highslide' folder into your plugins folder (/wp-content/plugins/).
2. Activate the plugin from the Plugins menu.

= Usage =

1. Click the Zoom-Highslide submenu to create you favarite setting.

= Changing the CSS =

If you need to configure the CSS style of Zoom-Highslide, you can edit highslide.css file directly.
